# Etapa_intermediaria
Entrega BootCamp II

## Como executar

1. Instale dependências:

```bash
npm install
```

2. Inicie o servidor:

```bash
npm start
```

3. Abra no navegador:

```text
http://localhost:3000
```

## O que foi adicionado

- `server.js`: servidor Express que expõe a API em `/api/games`
- `package.json`: dependência do Express e script `npm start`
- `index.html`: consumo da API local com `fetch` e renderização dinâmica dos jogos
